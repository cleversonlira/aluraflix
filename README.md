# Aluraflix Backend

Projeto do Challange da Alura de Back End

# ESCOPO

Realizar o desenvolvimento de Apis REST para disponibizar a possibilidade de troca de informações com outros sistemas.
O foco será implementar apis que irão manipular as informações sobre videos do youtube, afim de armazenar os videos que são mais interresantes.

# Ambiente para trabalhar neste projeto:

  Java - Para rodar o projeto em Back End, utilize sua IDE de preferência, importe o "Projeto Maven Existente".
  
  MySql Para rodar o Banco de Dados (Recomendo usar o Workbench)

# Como o Projeto Funciona?

O projeto é feito em Java + Spring, em conjunto com o Banco de Dados MySql.

MySql --> A biblioteca MySql é usada para se comunicar com um banco de dados em MySql.
